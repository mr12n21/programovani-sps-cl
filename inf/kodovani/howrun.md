# Jak spustit projekt

## Požadavky
- GCC kompilator  
- Nainstalovany `make`  

---

## Postup
V terminalu spust:

```bash
cd kodovani
make
make run
```
po kompilaci:
```bash
./kodovani
```

## Info
Program umoznuje:
    prevod textu do binárního kodu
    preklad do Morseovy abecedy
    kompresi pomoci Huffmanova kodovani
